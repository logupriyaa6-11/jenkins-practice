## commands
cd — Change Directory
ls – Lists files and directories in the current directory
ls -l – Displays files and directories in long format with detailed information
ls -a – Lists all files, including hidden files
ls -ld – Shows detailed information about the directory itself, not its contents
ls -t – Lists files sorted by modification time
ls -s – Displays file sizes along with file names

cp – Copies files or directories from one location to another
cp -a – Copies files and directories while preserving all attributes (permissions, ownership, timestamps, and symbolic links)
cp -r – Copies directories and their contents recursively
cp -i – Prompts before overwriting existing files
cp -v – Displays detailed output while copying
cp -p – Preserves file attributes such as permissions and timestamps
cp -u – Copies files only when the source is newer than the destination

mv – Moves or renames files and directories
rm – Deletes files or directories
echo – Displays text or variable values to standard output
touch – Creates empty files or updates file timestamps
cat – Displays the contents of files
less – Views file contents one page at a time with backward and forward navigation
more – Views file contents one page at a time with forward-only navigation
mkdir – Creates new directories
man + help – Displays documentation and usage information for commands
head – Shows the beginning portion of a file
tail – Shows the ending portion of a file
